# Corpus seed (illustratif)

Petit corpus **illustratif** pour bâtir et tester le pipeline RAG (Phase 3). Les champs
`source` sont des identifiants illustratifs (`seed:...`), **pas** de vraies citations.
L'ingestion de données réellement sourcées (World Bank / SIPRI / GDELT) est la **Phase 4**
(bascule Cowork). Format de chaque document : `{id, title, source, date, text}`.
